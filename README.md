<h3>Project Overview</h3>
<p>
A modern, interactive portfolio website showcasing security engineering leadership experience, built with a focus on performance and user experience. Features include an interactive photo component with spotlight effects, responsive design, and comprehensive work history in insider threat detection and incident response.
</p>

<h3>Technical Architecture</h3>
<p>
CRGOMEZ is built using <a href="https://aws.amazon.com/amplify/">AWS Amplify Hosting</a>, providing a fully managed CI/CD service for fast, secure, and reliable static serverless web apps.
</p>

<h3>Key Features</h3>
<ul>
<li><strong>Interactive Photo Component</strong> - Hover-activated spotlight effect revealing clear image through blurred background</li>
<li><strong>Responsive Two-Column Layout</strong> - Hero section with content on left, photo on right for desktop; stacked on mobile</li>
<li><strong>Modern Design System</strong> - Modular CSS architecture with design tokens, components, and utilities</li>
<li><strong>Project Detail Expansion</strong> - Clickable buttons revealing technical implementation details for work experience</li>
<li><strong>Flat Card Design</strong> - Consistent visual hierarchy throughout the portfolio</li>
</ul>

<h3>Why AWS Amplify?</h3>
<p>
AWS Amplify supports static sites, environment staging, and atomic deployments. Since
<a href="https://crgomez.com">CRGOMEZ</a> is meant to share personal career artifacts, having a service that supports continuous integration and continuous deployment (CI/CD) is essential.

The global network with instant cache invalidations ensures the site updates immediately on every GitHub code commit.
</p>
